# 产品需求

## 核心场景

家里有很多周期性琐碎事项，比如每 3 天铲猫砂、每月 1 日清洁马桶、每周日换床单。任务需要分配给不同家庭成员，并在手机 H5 App 和家中多个固定屏幕上同步展示。

## 必须功能

- 创建家庭空间，邀请家庭成员。
- 创建周期任务，支持按天、按周、按月日期、每月第几个星期几等规则。
- 给任务分配负责人，可单人或多人。
- 自动计算今天到期的任务。
- 手机端 H5 包装成 APK 后查看今天任务，并标记完成。
- 多个 ESP32 屏幕绑定同一家庭，通过 Wi-Fi 同步今天任务。
- ESP 设备端直接访问专用 H5 链接，展示今天每个人要做的事情。
- 设备端支持极简操作：点任务标记完成。
- 离线或网络不稳定时，设备端保留最后一次成功同步的今日列表。

## 可选功能

- 米家/小爱音箱查询今天任务。
- 语音标记完成。
- 定时提醒，例如每天早上 8 点播报未完成事项。
- 家庭成员积分、连续完成记录、任务负载均衡。

## MVP 范围

1. H5 手机端：成员、任务、今日列表、完成状态，并可包装成 APK。
2. Firebase：登录、家庭空间、任务、完成记录、设备绑定。
3. ESP32：Wi-Fi 配网、保存设备链接或 token、加载设备 H5 页面、触摸完成。
4. 阿里云：部署轻量 API，用于设备 token 签发、语音集成 webhook、定时任务。

## 关键产品判断

- 设备端使用独立 H5 页面，不做复杂管理，只做查看和完成。
- 完成记录按日期独立保存，周期任务本身不因完成而改写。
- 任务到期计算优先在服务端完成，设备页面只消费简化后的今日列表。
- 家庭空间是权限边界，设备只属于一个家庭。
